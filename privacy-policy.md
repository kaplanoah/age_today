# Privacy Policy — Age Today

**Last updated:** March 8, 2026

Age Today is a Chrome extension that calculates a person's current age from a birth date.

## Data Collection

Age Today does **not** collect, store, transmit, or share any personal data or user information.

## Permissions

- **activeTab / scripting**: Used solely to read text you have selected on the current page when you open the extension. This data is only used locally within the popup to pre-fill the date input. It is never stored or sent anywhere.

## Third Parties

Age Today does not communicate with any external servers or third-party services. All processing happens locally in your browser.

## Changes

If this policy is updated, the changes will be reflected on this page with an updated date.

## Contact

If you have questions about this policy, please open an issue at https://github.com/kaplanoah/age_today/issues.
